# ITSx
ITSx - ITS extractor
## Prep
```
curl -O http://microbiology.se/sw/ITSx_1.0.11.tar.gz
```

## Compile

## INSTALL
```
rsync -a ITSx /opt/linux/centos/7.x/x86_64/pkgs/ITSx/1.0.11/bin
rsync -a ITSx_db /opt/linux/centos/7.x/x86_64/pkgs/ITSx/1.0.11/share
```

Nothing special.
